# ♟️ Šaha apmācības programma

## Par programmu
Šī ir vienkārša šaha apmācības programma diviem spēlētājiem (White un Black).

Programma ļauj:
- pārvietot figūras uz šaha laukuma
- ievadīt gājienus koordinātēs (piemēram, e2 → e4)
- redzēt spēles laukumu konsolē

---

## Noteikumi
- Spēlē piedalās 2 spēlētāji: White un Black
- Katrs spēlētājs veic vienu gājienu pēc kārtas
- Gājieni tiek ievadīti formātā:
  - No: e2
  - Uz: e4
- Programma nepārbauda pilnus šaha noteikumus (tikai pārvietošanu)
